# Cyclistic Case Study

This analsysis is from the Cyclistic Bike Share Case Study: 'How Does a Bike-Share Navigate Speedy Success?' and  was done as part of the Google Data Analytics Certificate 
Capstone Project offered through Coursera.com. The data used in this case study was part of the 2022 monthly Divvy trip data set and was made available for public use by the 
Motivate International Inc. license
